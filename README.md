# 3D Minimalist Grid

A clean, distraction-free $36 \times 36 \times 36$ 3D grid environment built with Three.js.

## Features
- **Interactive Perspective:** Drag to rotate, scroll to zoom, and right-click to pan.
- **Minimalist Aesthetic:** High-contrast dark mode with subtle wireframe accents.
- **Responsive:** Automatically adjusts to any screen size or mobile orientation.

## Setup
Simply host these files on GitHub Pages or use a local live server (like VS Code Live Server extension) to view the project.
